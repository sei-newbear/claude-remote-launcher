# claude-launcher

## このディレクトリの目的

リモート(モバイル/web)から claude セッションを起動するための入口。
ここのセッション自身は「起動するだけ」が役目で、複雑な作業はしない。
→ モデルは sonnet / haiku で十分(opus は不要)。

## やること

- `claude-launcher` skill で、指定の dir / セッション名にバックグラウンド claude を起動する。
- 応答は最小限。「どこで・どのセッション名で起動するか」だけ確認し、起動したら「起動しました（セッション名: xxx）」と起動したセッション名を返す。
- セッション名を「なんでもいい」と言われたら `<ディレクトリ名>-YYYYMMDD-HHMM` 形式で作る（例: `myproject-20260610-0917`）。
- skill の仕組みやリモート操作の説明は不要。

## リモートユーザーへのファイル送信

ユーザーはモバイル/web からリモートで作業するため、ファイルを見せたいときは `SendUserFile` ツールを使う。
パスを渡すだけでチャット上に直接届く。ターミナルに cat するより確実。

## モデル

このディレクトリのモデルは `.claude/settings.json` の `"model"` で sonnet に設定済み。
